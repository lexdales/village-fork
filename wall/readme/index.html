A) Intro - top

Wall Media Gallery - jQuery poweredis a modern design gallery for your pictures/ swfs/ videos/ YouTube videos. It can be used as a portfolio or just as your cool gallery for your work. Wall Media Gallery is easy to use and very customizable. All these features are brought together in a very different style

B) Installation - top

In order to install this component in your html page you need to:
a) open the download package, copy the folder \js , the folder \img and the folder \style into your html page folder
b) in the head tag of your html page write

<script src="http://code.jquery.com/jquery-1.5.1.min.js"></script> <script src="js/shiftGallery.js"></script>
c) in the body tag of your html write

--if you want to add a preloader add this line in your html
<div id="preloader"><img src="preloader.gif"/>Loading...</div>

<div id="wall">
<img src="img/1.jpg"/>
<a href="img/1.jpg" rel="prettyPhoto"><img src="img/1.jpg"/></a>
<img title="description" src="img/1.jpg"/>
<div >content with text</div>
</div>

d) before the end of your body tag add this (#yourId is the id of your div)

<script>
jQuery(document).ready(function(){
$('#wall').wall({
settings_thumbs_per_row:5,
settings_width:0,
settings_height:0,
thumb_width:550,
thumb_height:300,
thumb_space:2,
settings_padding:20,
description_height:30
})
$("a[rel^='prettyPhoto']").prettyPhoto();
})
</script>


e) that is all, now you just have to replace the text from src with your images. If you get confused, just view the source of this html on which the gallery is setup.

C) Customising your Wall - top

<div id="wall">
In your div with "wall" id you will add wall components
<img src="img/2.jpg"/>

This is how you add a simple image

<img title="here goes your description" src="img/2.jpg"/>

This is how you add a image with a description
<a href="img/1.jpg" rel="prettyPhoto"><img src="img/1.jpg"/></a>

This is how you add a picture width lightbox effect ( href="img/1.jpg" -- here is the source of the image, wich will appear in lightbox)

<a href="http://www.youtube.com/watch?v=N8U2xo1WkXM" rel="prettyPhoto"><img src="img/7.jpg"/></a>
This is how you insert a youtube video in a lightbox
<div >content with text</div>

Just insert a div with content that you want it .

<div>
<img class="p1" src="img/s5.jpg"/>
<img class="p2" src="img/s6.jpg"/>
<img clas="p3" src="img/s7.jpg"/>
</div>
THIS IS ONLY IF YOU CHOOSE thumb_width:550 AND thumb_height:300

 

This is the css for these images:

.p1{
width:33%;
height:100%;
float:left;
position:relative!important;
}
.p2{
left:185px;
}
.p3{
left:185px;
top:151px;
}

This is how you add multiple images into a single div.
In this example (thumb_width:550 thumb_height:300)

  • s5 image is 183 x 300 px
  • s6 and s7 are 365 x 149 px

CUSTOMIZABLE VARIABLES: 

  • settings_thumbs_per_row – number of components of the wall that will appear on a line
  • settings_width - wall width limit, if you want to have one ( if not put 0)
  • settings_height - wall height limit
  • thumb_space - the space between window and wall
  • settings_padding - the space between window and wall
  • description_height - background height description (px)

How can we put the wall into a small frame?

Just modify the main div <div id="wall" style="width:600px; height:600px;"> to desired width and height.(in this case 600 x 600 px)
